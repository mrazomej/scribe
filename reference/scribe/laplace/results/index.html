    results ¶

    Results container for Laplace-mode inference.

    This module defines :class:ScribeLaplaceResults, the single public results class returned by Laplace inference for every supported base model. The class is a dataclass carrying fitted state and diagnostics, while behavior is implemented via mixins grouped by responsibility (core accessors, model dispatch, sampling, likelihood evaluation, gene subsetting, and serialization).

    Results from any Laplace-mode fit.

    PARAMETER DESCRIPTION
    model_config

    Configuration used during fitting and dispatch. The base_model field determines runtime branch selection in dispatching methods.

    TYPE: ModelConfig or None

    mu

    Decoder bias vector in model-specific latent coordinates:

    • PLN: shape (G,) in log-rate coordinates.
    • LNM/LNMVCP: shape (G-1,) in ALR coordinates.

    TYPE: ndarray

    W

    Decoder loading matrix with shape (G, k) for PLN or (G-1, k) for LNM-family models.

    TYPE: ndarray

    d

    Positive diagonal residual variance aligned with mu.

    TYPE: ndarray

    final_grad_norms

    Final per-cell Newton gradient diagnostics.

    TYPE: ndarray

    losses

    Outer-loop objective history.

    TYPE: ndarray

    n_genes

    Number of genes represented by this result.

    TYPE: int

    n_cells

    Number of cells represented by this result.

    TYPE: int

    x_loc

    Model-specific per-cell MAP latent slots. Exactly one latent branch is typically populated:

    • PLN: x_loc (and optionally eta_loc for capture-anchor fits)
    • LNM low-rank: z_loc
    • LNM learned-diagonal: y_alr_loc
    • LNMVCP: one of the LNM slots plus p_capture_loc

    TYPE: ndarray DEFAULT: None

    eta_loc

    Model-specific per-cell MAP latent slots. Exactly one latent branch is typically populated:

    • PLN: x_loc (and optionally eta_loc for capture-anchor fits)
    • LNM low-rank: z_loc
    • LNM learned-diagonal: y_alr_loc
    • LNMVCP: one of the LNM slots plus p_capture_loc

    TYPE: ndarray DEFAULT: None

    z_loc

    Model-specific per-cell MAP latent slots. Exactly one latent branch is typically populated:

    • PLN: x_loc (and optionally eta_loc for capture-anchor fits)
    • LNM low-rank: z_loc
    • LNM learned-diagonal: y_alr_loc
    • LNMVCP: one of the LNM slots plus p_capture_loc

    TYPE: ndarray DEFAULT: None

    y_alr_loc

    Model-specific per-cell MAP latent slots. Exactly one latent branch is typically populated:

    • PLN: x_loc (and optionally eta_loc for capture-anchor fits)
    • LNM low-rank: z_loc
    • LNM learned-diagonal: y_alr_loc
    • LNMVCP: one of the LNM slots plus p_capture_loc

    TYPE: ndarray DEFAULT: None

    p_capture_loc

    Model-specific per-cell MAP latent slots. Exactly one latent branch is typically populated:

    • PLN: x_loc (and optionally eta_loc for capture-anchor fits)
    • LNM low-rank: z_loc
    • LNM learned-diagonal: y_alr_loc
    • LNMVCP: one of the LNM slots plus p_capture_loc

    TYPE: ndarray DEFAULT: None

    alr_reference_idx

    ALR reference gene index for LNM-family models.

    TYPE: int DEFAULT: None

    TYPE: ndarray DEFAULT: None

    TYPE: ndarray DEFAULT: None

    mu_T_scale

    TYPE: ndarray DEFAULT: None

    r_T_loc

    Unconstrained LNM totals marginal posterior parameters. Scalar arrays.

    TYPE: ndarray DEFAULT: None

    r_T_scale

    Unconstrained LNM totals marginal posterior parameters. Scalar arrays.

    TYPE: ndarray DEFAULT: None

    totals_cov

    Full 2x2 covariance for the joint (mu_T, r_T) unconstrained posterior. Important for deriving the distribution of the success probability p = r_T / (r_T + mu_T).

    var

    AnnData-style metadata.

    TYPE: Any DEFAULT: None

    obs

    AnnData-style metadata.

    TYPE: Any DEFAULT: None

    n_obs

    Source-data shape metadata.

    TYPE: int DEFAULT: None

    n_vars

    Source-data shape metadata.

    TYPE: int DEFAULT: None

    early_stopped

    Whether early stopping was triggered.

    TYPE: bool DEFAULT: False

    best_loss

    Best smoothed loss observed during optimization.

    TYPE: float DEFAULT: inf

    stopped_at_step

    Final optimization step.

    TYPE: int DEFAULT: 0

    divergence_aborted

    Whether divergence detection aborted training.

    TYPE: bool DEFAULT: False

    _subset_gene_index

    Internal bookkeeping for gene subset views.

    TYPE: ndarray DEFAULT: None

    metadata

    Free-form metadata and compatibility storage used by plotting helpers (for example predictive_samples and posterior_samples caches).

    TYPE: dict, default_factory=dict DEFAULT: dict()

    frozen_params

    Set of NBLN parameter short-names {"r", "mu", "eta"} that were pinned at the SVI-cascade-source MAP during the Laplace M-step (see informative_priors_freeze on scribe.fit). Metadata only: indicates provenance of the corresponding posterior — when a key is present here, get_distributions() and PPC paths route through cascade_source rather than the NBLN profiled Hessian.

    TYPE: frozenset[str] DEFAULT: empty

    cascade_source

    Source SVI results object embedded by reference when a freeze cascade was active. Used to recover full-fidelity posterior samples and moment-matched distributions for any parameter listed in frozen_params. None for non-cascade fits.

    TYPE: ScribeSVIResults DEFAULT: None

    cascade_source_counts

    Cached count matrix written when the embedded cascade source uses amortized capture and its own _original_counts is unset, so that PPC sampling can evaluate the amortized encoder after pickle. Set from ctx.count_data at bridge time; never mutates the source.

    TYPE: ndarray DEFAULT: None

    w_prior_diagnostics

    Per-fit summary of the loadings-shrinkage prior — keys include strategy_type, sigma_k, column_frobenius_compositional (the gauge-invariant per-factor norm — the headline rank diagnostic), column_frobenius_raw, column_norm_effective_rank / effective_rank, and strategy-specific aux MAPs (e.g. tau for horseshoe, gamma for NEG). None when no shrinkage prior was active. See scribe.viz.plot_w_shrinkage_spectrum.

    TYPE: dict DEFAULT: None

    Notes

    The class is intentionally model-agnostic at the type level: all Laplace results share one container type and dispatch internally by base_model. This mirrors the high-level SCRIBE design of one result class per inference method.

    See Also

    scribe.svi.results.ScribeSVIResults SVI results class that uses the same mixin-composition architecture.

    def __repr__(self) -> str:
    """Return a concise text summary for interactive inspection.

    The dataclass default representation recursively expands all fields,
    including large arrays and metadata dictionaries. This compact repr
    keeps notebook output readable by showing only high-value summary
    fields and lightweight status flags.

    Returns
    -------
    str
        Stable summary string with model identity, dataset shape,
        optimization history length, latent-state branch, and global
        uncertainty status.
    """
    n_steps = int(len(self.losses)) if self.losses is not None else 0
    base_model = self._base_model_name()
    latent_state = self._summarize_latent_state()
    uncertainty = self._summarize_global_uncertainty()
    return (
        "ScribeLaplaceResults("
        f"model={base_model!r}, "
        f"n_cells={self.n_cells}, "
        f"n_genes={self.n_genes}, "
        f"n_steps={n_steps}, "
        f"latent={latent_state!r}, "
        f"uncertainty={uncertainty!r})"
    )
